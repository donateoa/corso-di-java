# CONTEXT FOR CODEX

Questo progetto contiene una app Java chiamata `SpringMockExam.java` che simula l’esame Spring Certification con:

- domande singole e multiple
- scoring in tempo reale
- percentuale
- riepilogo finale

Obiettivo: migliorare l’app, aggiungere domande, validazioni, funzionalità aggiuntive o esportazione JSON.

Regole:

- Mantieni compatibilità con Java 17+.
- Non introdurre buchi di sicurezza.
- Aggiungere nuova funzionalità **senza rompere il flusso esistente**.
- Evitare side effects globali.
